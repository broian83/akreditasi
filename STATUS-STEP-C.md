### Langkah 1: Daftar Mayar
1. Buka https://mayar.id
2. Daftar & verifikasi KTP
3. Tunggu approval (1-2 hari)

### Langkah 2: Buat 3 Produk
| Produk | Harga | Link |
|--------|-------|------|
| Basic | Rp 149K | Buat di Mayar |
| Pro | Rp 299K | Buat di Mayar |
| Premium | Rp 499K | Buat di Mayar |

### Langkah 3: Update Links
Ganti di `LANDING-PAGE/index.html`:
```html
<!-- Ganti ini -->
https://mayar.id/checkout/your-product-id-basic

<!-- Dengan -->
https://mayar.id/checkout/ABC123-basic
```

### Langkah 4: Setup Delivery
1. Upload file template ke produk Mayar
2. Set delivery = Instant
3. Customer bayar → langsung dapat link download
4. Selesai! ✅
