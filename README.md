# 🏥 Template Akreditasi RS - Google Apps Script

Template Google Sheets + Apps Script untuk tracking kelengkapan dokumen akreditasi Rumah Sakit.

## 📦 Fitur
- ✅ Dashboard progress akreditasi
- ✅ Tracking dokumen per standar/kriteria
- ✅ Assignment PIC & deadline
- ✅ Reminder otomatis
- ✅ Export laporan
- ✅ Multi-user collaboration

## 🚀 Cara Install
1. Buka link template Google Sheets
2. Klik "Make a copy" 
3. Ikuti panduan setup di PDF
4. Deploy sebagai Web App
5. Selesai!
