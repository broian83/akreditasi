## 🔧 TROUBLESHOOTING

### Menu "🏥 Akreditasi RS" tidak muncul?
**Solusi:**
1. Di Apps Script Editor, klik "Run" > "onOpen"
2. Authorize jika diminta
3. Refresh Google Sheets
4. Tunggu 5 detik

### Web App error "Script function not found"?
**Solusi:**
1. Pastikan `doGet()` function ada di Code.gs
2. Re-deploy: `clasp deploy` lagi

### Sheets belum ter-setup?
**Solusi:**
1. Buka Google Sheets
2. Klik menu "🏥 Akreditasi RS" > "Setup Awal"
3. Atau run `setupSheets()` dari Apps Script Editor
