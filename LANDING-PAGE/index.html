Pertanyaan yang Sering Ditanyakan ❓

Q: Apakah bisa dipakai untuk multiple RS?

A: Tidak. Setiap RS harus punya copy template sendiri (via "Make a copy" di Google Sheets).

Q: Apakah ada batasan jumlah dokumen?

A: Tidak ada. Google Sheets bisa handle 10,000+ baris dokumen.

Q: Apakah data aman?

A: Ya, 100% aman. Data tersimpan di Google Drive Anda sendiri, bukan di server pihak ketiga.

Q: Bisa akses dari HP?

A: Bisa! Google Sheets dan Web App bisa diakses dari browser HP dengan responsive design.

Q: Apakah perlu install aplikasi tambahan?

A: Tidak. Cukup Google Sheets + browser untuk Web App. No install required!

Q: Fonnte gratis?

A: Ya, FREE 1000 pesan/bulan. Cukup untuk RS rata-rata. Jika lebih, bisa upgrade ke paket berbayar.
