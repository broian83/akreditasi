## 🎯 CARA PAKAI TEMPLATE

### Langkah 1: Pilih Template
Pilih dokumen sesuai kebutuhan dari daftar di atas

### Langkah 2: Download/Copy
Copy isi file markdown ke Google Docs/Word

### Langkah 3: Customize
Sesuaikan dengan kondisi RS Anda:
- Ganti [Tanggal], [Nama] dengan data RS
- Sesuaikan kebijakan dengan kondisi lokal
- Tambah logo & kop surat RS
- Modifikasi prosedur sesuai SOP existing

### Langkah 4: Review
Review oleh:
- Kepala Ruangan/Kepala Instalasi
- Komite terkait
- Direktur/Direktur Medik

### Langkah 5: Implementasi
- Upload ke tracking system
- Sosialisasikan ke staff
- Monitoring kepatuhan
